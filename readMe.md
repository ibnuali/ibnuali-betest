
## API Reference

### Endpoint User
method   | url                  | Description                       |
:------- |:----------------     | :-------------------------------- |
`GET`    | `/api/v1/user`     | get all users               |
`GET`    | `/api/v1/user/:id` | get all data user by id         |
`POST`   | `/api/v1/user`     | create data user                |
`PUT`    | `/api/v1/user/:id` | edit data user                  |
`DELETE` | `/api/v1/user/:id` | delete data user                |
